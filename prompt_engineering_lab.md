# Prompt Engineering Lab – Meshack Jr

## Overview
This project demonstrates how different prompt styles affect AI output quality.

---

## 1. Bad Prompt Example
"Tell me about AI"

### Result:
Generic, unstructured, low-quality response.

---

## 2. Improved Prompt Example
"Explain Artificial Intelligence in 5 bullet points for a beginner, including real-world examples."

### Result:
- Structured response
- Clear formatting
- Easier to understand

---

## 3. Advanced Prompt Example
"Act as a senior AI engineer. Explain how a machine learning model works using simple analogies, then give a technical breakdown."

### Result:
- Dual-level explanation (beginner + advanced)
- Context-aware response
- Professional AI-style output

---

## Key Learning
Better prompts = better AI outputs.

Prompt engineering is a core skill in modern AI workflows.
